Lesson 1

The difference between
HTML and CSS

Before you actually start learning how to code in HTML and CSS, you have to know what HTML and CSS stand for and their role in websites.

HTML is short for "Hyper Text Markup Language" and contains the content of the webpage. See this paragraph you are reading now? It is typed into the HTML of this website.

CSS, on the other hand, is short for "Cascading Style Sheet" and is used to make the webpage look nice ("styling" it). Why is there a bar at the right-hand side. Why is the header green? This is all controlled by CSS.

Now, you may ask: it is necessary to use both HTML and CSS when coding a website? The answer is: no and yes. To build a website, only HTML is required, but websites made purely by HTML look like word documents (hardly attractive!) - except that you can't even edit it! CSS, on the other hand, cannot stand alone - if you use CSS, you must also use HTML.
