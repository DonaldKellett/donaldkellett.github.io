Now that I know a bit more about HTML5 Geolocation, I am able to make a slightly more sophisticated app that tracks the user's location. gpsNav - The app for online GPS tracking

This is my very first MIT Licensed application!

If you (or your company, or whatever) own a website and want to install some type of GPS tracking device, but are struggling to do so, just download this app! It is user-friendly, the setup code (not the source code) is human-readable and it is dead easy to customize it according to your own needs. Just go to Donald Leung - Code and scroll down to "Licensed Projects" (a whole new category!) to read more about this amazing app! A gh-pages version will be available soon.
