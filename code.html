This knowledge has allowed me to make a primitive search engine that works locally in a website. However, the search engine is not very efficient, because if you add a lot of keywords, a lot of irrelevant results will appear and every time a new page in the site is added, the search engine has to be updated manually.

HTML Display on Browser

I have finally learned how to display HTML code on the browser. Displaying HTML on the browser is different (and much more difficult) from displaying other types of code on the browser because if you simply type in html code in the <code> tags, the file will simply read it as HTML tags and it will not display in the browser. See if you can figure out how to do this (no cheating!) without throwing the file in the text editor to read all the HTML code.
