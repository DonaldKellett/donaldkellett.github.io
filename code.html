Internal Search Engine (Prototype)

Please note that this project is also copyrighted. You can freely download it for your own use, but if you make it public, please credit me by adding the following: Internal Search Engine by Donald Leung

I have recently learned a function that allows me to split strings into different pieces using a specific sequence of characters as the 'delimiter'. This knowledge has allowed me to make a primitive search engine that works locally in a website. However, the search engine is not very efficient, because if you add a lot of keywords, a lot of irrelevant results will appear and every time a new page in the site is added, the search engine has to be updated manually.
